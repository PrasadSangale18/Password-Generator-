# 🔐 Password Generator

A simple Python CLI tool that generates strong, customizable passwords with a strength checker.

---

## 🚀 Features

- Customizable length
- Toggle uppercase, digits, symbols
- Generate multiple passwords at once
- Built-in strength checker (Weak / Medium / Strong / Very Strong)
- Zero dependencies — uses Python standard library only

---

## ⚙️ Run

```bash
python password_generator.py
```
